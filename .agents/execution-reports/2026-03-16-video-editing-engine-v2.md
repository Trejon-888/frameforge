## What Was Built/Changed

- `video-probe.ts` — FFprobe wrapper for reading source video metadata (resolution, bitrate, codec, audio)
- `word-captions.ts` — Word-level caption engine with grouping algorithm and 5 presets (pop-in, karaoke, highlight, minimal, bold-center)
- `edit-styles.ts` — Style preset system with 4 presets (neo-brutalist, clean-minimal, corporate, bold-dark)
- `overlay-generator.ts` — Content analysis engine that auto-detects hooks, introductions, stats, key points, topic transitions from transcript
- `editor.ts` — Main orchestrator (probe → transcribe → analyze → generate overlay → render → mux audio)
- `cli.ts` — New `frameforge edit` command with 14 CLI options
- `ffmpeg.ts` — Updated with configurable CRF, preset, maxrate, bufsize, audioBitrate
- 4 new test files with 57 tests
