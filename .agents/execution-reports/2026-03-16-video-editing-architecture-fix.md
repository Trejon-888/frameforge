## Architecture Decision

**Never decode source video through a browser for editing.** Browser video element seeking is not frame-accurate. Instead: render overlay as transparent PNG frames via Puppeteer, composite onto source video via FFmpeg's native `overlay` filter. Source video stays in FFmpeg's domain = smooth playback.
