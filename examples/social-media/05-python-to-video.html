Python SDK

Python to Video

No JavaScript required. Define scenes in Python, render to MP4.

render.py
from frameforge import Scene, Text, Shape
scene = Scene(duration=5, background="#0a0a0a")
title = Text("Hello World",
font_size=72, color="#ffffff")
title.animate("opacity", {0: 0, 1: 1})
scene.add(title)
scene.render("output.mp4") # → MP4
